# Hybrid Encryption System (AES + RSA)

## Description
This project demonstrates the implementation of symmetric and asymmetric encryption algorithms, specifically AES and RSA. The objective is to understand how modern cryptographic systems secure data and manage keys.

## Objectives
- Implement AES (Advanced Encryption Standard) for symmetric encryption.
- Implement RSA for asymmetric encryption.
- Explore key generation, encryption, and decryption processes.
- Understand the differences between symmetric and asymmetric cryptography.

## Technologies Used
- Python
- Basic cryptographic libraries 

## Key Learning Outcomes
- Practical understanding of encryption algorithms.
- Hands-on experience with secure key handling.
- Comparison between classical cryptography and modern secure communication models.
